# 🧮 Scientific Calculator (Vivid Calc)

A modern and advanced Scientific Calculator built using HTML, CSS, and JavaScript. It supports complex mathematical operations like trigonometry, logarithms, factorials, and power functions with a smooth UI and real-time calculations.

---

## 🚀 Features

- 🔢 Basic arithmetic operations (+, −, ×, ÷)  
- 📐 Trigonometric functions (sin, cos, tan)  
- 🔄 Degree / Radian mode toggle  
- 🔁 Inverse functions (asin, acos, atan)  
- 📊 Logarithmic functions (log, ln)  
- 🧮 Square root, power (^), factorial (!)  
- 📌 Constants (π, e)  
- 📜 Calculation history panel  
- 🎨 Modern colorful UI  
- ⌨️ Keyboard support  
- ⚡ Fast and accurate evaluation (custom parser, no eval)

## 🎯 Usage

1. Enter numbers using buttons or keyboard  
2. Use operators for calculations  
3. Enable **Sci mode** for advanced functions  
4. Toggle:
   - Deg / Rad for angle units  
   - Inv for inverse functions  
5. Press "=" to evaluate  
6. View previous calculations in History
